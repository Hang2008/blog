# 被 vuepress 坑

# 填坑项目和办法

  • 使用自定义主题
  1. 官方项目默认安装了@vuepress/theme-blog, 可以卸载重装
  2. 找到目录@vuepress😐, 复制 theme-blog 目录到.vuepress 中
  3. 重命名文件夹为 theme
  4. 修改 config
// theme: "@vuepress/theme-blog", // OR shortcut: @vuepress/blogƒ
  1. 修改样式
  • 官方项目 tag 页面显示空白 修改所有 markdown 中的tagtags:
---
date: 2020-4-04
tags:
  - hero
author: 亢亢兮
location: 金陵
---
