entryComponents 是什么? A: 定义此 NgModule 中要编译的组件集，这样它们才可以动态加载到视图中。

响应式表单和模板驱动表单有什么区别?

不过，响应式表单和模板驱动表单在如何处理和管理表单和表单数据方面有所不同。各有优势。

A: 响应式就是要引ReactiveFormsModule这个模块的技术，不管是模型触发视图，还是视图触发模型，都是由FormControl实例通过valueChanges可观察对象发出新值，是一个中间桥梁，ControlValueAccessor作为更新的借口。模板驱动无非就是用双向数据绑定在其中作为数据变化机制。

# 性能优化总结

组件库开发: 开发具有And-D风格的项目公共业务组件。银行系统业务非常复杂，需要自定义满足各类需求的业务组件。

项目优化: 不要写复杂的html页面，只通过写简单的业务逻辑实现页面自动生成

项目优化: 封装业务库，通过编写JSON文件实现具有业务逻辑的页面， 省去繁琐重复的编码工作

# tips

  • ngOnChanges 在组件初始化的时候会调用一次，后面判断是否第一次调用可以用变量的firstChange属性
if (changes.productDetails && !changes.productDetails.firstChange && changes.productDetails.currentValue) {}
  • ngOnChanges只有在输入属性被改变时才会调用
  • RxJS 属于响应式编程，其思想是将时间看作数组，随着时间发生的事件被看作是数组的项，然后以操作数组的方式变换事件。其强大的地方在于站在四维的角度看问题，这就像是拥有了上帝视野。
